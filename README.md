# h5-guessscore-notes

## 简介

本仓库用于归档和发布多个独立的 HTML 页面。  
这些页面以静态网页形式存在，不针对任何特定域名或网站，仅作为通用 HTML 存档与展示用途。

## 目录说明

```
h5-guessscore-notes/
├── index.html          # 仓库总索引页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，彼此无依赖关系。
- `assets/`：存放页面共同引用的样式、脚本或媒体文件。

## 页面归档说明

- 所有页面均为静态 HTML，可直接通过浏览器打开。
- 页面内容不涉及具体商业或推广信息，仅作为技术演示或资料存档。
- 每个页面文件均有独立命名，便于索引与引用。

## 维护说明

- 本仓库由作者不定期更新，新增或修改页面时会保持目录结构清晰。
- 欢迎通过 Issues 提交建议或报告页面显示问题。
- 不设自动构建或部署流程，页面以原始 HTML 形式直接托管。
